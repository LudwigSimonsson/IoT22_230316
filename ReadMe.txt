Denna filsamling är riktad till IoT22 vid Nackademins labbdag 2023-03-16..
Målet idag är att kartlägga vart ni befinner er kompetensmässigt när det kommer till C++-språket.
Materialet skall genomarbetas på följande sätt:

Inled med att besvara följande formulär:
https://forms.office.com/e/ZvwjkjawvD

Gå sedan vidare till detta;
Hämta repot med samtliga filer. Öppna det sedan i VS Code. Se till att ha installerat Code Runner samt nödvändiga paket för C++.
Inled sedan arbetet nedan:

- Under C++_General så finner ni beskrivningar och exempel på C++ kod för att jämföra vid eventuella problem
- Under C++_Tasks så finner ni 48 små utmaningar som ni skall prioritera för att uppvisa er kunskap inom C/C++
- Under C++_TaskSolutions så finner ni lösningar för samtliga 48 övningar så att ni kan säkerställa att ni utfört som tänkt
- Efter man klarat av dessa utmaningar så vänder man sig till C++_Controls som innehåller elva bredare övningar för genomförande
- Under C++_ControlSolutions så finner ni lösningarna för dessa 11 problem för att kunna rätta ert egna arbete
Om samtliga av dessa 59 uppgifter hinns med så kan man vidaregå till C++_Final som är en uppgift där man testas på samtliga C++-kunskaper. Detta motsvarar grundläggande examination av C++
Om även denna utmaning skulle hinnas med, skall man sedan studera C&C++_ADS där man finner ett hav av exempel på algoritmer och datastrukturer som är relevanta för utbildningen
